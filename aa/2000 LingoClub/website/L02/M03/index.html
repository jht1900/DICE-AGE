Level 2, Mission 3:

Creating Animation Cycles.

Use conbination of commands to produce animation using number cycles.

Following script will cycled through cast members 1, 2, 3. The pattern would be 1, 2, 3, 1, 2, 3, and so on.

Use If-then to keep a value from going beyond a certain limit.

Cast Member

sprite(1).memberNum = spriteNum(1).memberNum + 1
if sprite(1).memberNum > 3 then
    sprite(1).memberNum = 1
end if

Location:

sprite(1).locH = sprite(1).locH + 1
if sprite(1).locH > 200 then
    sprite(1).locH = 10
end if

Know The Lingo

Expressions can contain arithmetic (+ - * /) operations. Test can contain comparision operations.

Can use any number of commands in a script.

Nesting of if-then commands.
