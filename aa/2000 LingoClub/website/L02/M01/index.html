Level 2, Mission 1:

Changing Sprite Properties.

Use Lingo to change the appearance by setting sprite properties.

Changing Sprite Properties

sprite(1).locH = 10 -- changes the horizontal position
sprite(1).locV = 20 -- change the vertical position
sprite(1).memberNum = 3 -- change the cast member
sprite(1).memberNum = 0 -- no cast member, hides sprite
sprite(1).rotation = 90 -- change the rotation angle

sprite(1).color = rgb(0,0,0)
sprite(1).bgcolor = rgb(255,255,255)
sprite(1).flipH = 1
sprite(1).flipV = 1
sprite(1).loc = point(10,20)

Events

When will change happen?

on mouseUp
on mouseDown
on mouseEnter
on mouseLeave
on mouseWithin

on exitFrame
on beginSprite

Simple changes.

sprite(1).rotation = 90

Increment changes.

sprite(1).rotation = sprite(1).rotation + 1
